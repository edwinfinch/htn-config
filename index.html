Class Schedules

Instructions

To sync a class:
1. Check what class out of 6 it is, (ie. class 2/6) in the "class number" section. This number will accordingly insert the class into that slot on the watch.
2. Insert all of the details in the class in the "details" section
3. Send data to pebble
The app will figure out on its own what class is next for you.

Made a mistake? The settings page remembers the last synced class for you to edit again, and again, and again.
